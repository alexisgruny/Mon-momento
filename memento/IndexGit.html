GIT

Pour initialiser mon git je fais un git init

Pour ajouter les fichier voulu je fais un git add

pour enregistrer mes modification dans le depot je fais un git commit

je peu verifier mes modification avec un git log

je peu verifier que tout ce passe bien avec un git status

pour envoyer mes depots en ligne j'utilise un git push

pour cree une branche je fais un git branch

pour voyager entre mes branches j'utilise git checkout

pour fusionner 2 branches je fais un git merge

pour changer le repertoire actuel je fais un git remote
