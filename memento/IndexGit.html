GIT
commande description
git init Pour initialiser mon repertoire de git
git add Pour ajouter les fichier voulu
git commit pour enregistrer mes modification dans le depot
git log verifier mes modification
git status verifier que tout ce passe bien
git push envoyer mes depots en ligne
git branch cree une branche
git checkout voyager entre mes branches
git merge fusionne 2 branches
git remote changer le repertoire actuel
