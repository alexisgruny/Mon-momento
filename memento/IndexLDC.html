La ligne de commande

Je peu avoir la description de la commande demandé avec la commande : man

Savoir se déplacer/retrouver

Pour connaitre les dossier present dans le dossier actuel j'utilise la commande : ls

j'y ajoute : -a (pour les dossier caché) j'y ajoute :-l (pour avoir une version longue)

pour afficher le chemin du dossie courant j'utilise la commande : pwd

Pour me déplacer dans un fichier j'utilise la commande : cd

Gestion de documents

Pour créer un dossier je dois utiliser cette commande : mkdir (add -p pour crée un dossier avec un sous dossier)

Pour créer un fichier je dois utiliser cette commande : touch

Pour copier un fichier/dossier je dois utiliser cette commande : cp

Pour remove un dossier/fichier je dois utiliser cette commande : rm (add -r pour supprimmer un dossier)

pour editer un fichier avec atom j'utilise : atom + nom du fichier

pour déplacer un ficher/dossier j'utilise la commande : mv

gestion/installation de paquet

pour etre en mode super utilisateur j'utilise la commande : sudo

pour mettre a jours la listes de paquets j'utilise la commande : apt update

pour mettre a jours les paquets j'utilise la commande : apt upgrade

pour faire une recherche de paquets j'utilise la commande : apt search + la recherche

pour installer un paquet j'utilise la commande : apt install + nom du paquet

pour lancer l'installation du .deb precedement telecharge j'utilise la commande : dpkg + nom du paquet
