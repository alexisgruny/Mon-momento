HTML

Mise en page
balise description
title mettre un titre a notre onglet
h1 met un titre de niveau 1 (h2 de niveau 2 et ainsi de suite)
p crée un paragraphe
ul represente une liste d'element
li represente un element dans une liste
span permet de donner un attribut a une partie du texte sans incidence sur le reste
em permet de mettre la saisie en italique
strong permet de la mettre en gras
Les liens
balise description
img permet d'afficher une image
a defini un hyperlien vers un autre endroit
alt permet d'ajouter un texte alternatif a une image ou un lien
title permet de rajouter un titre a une image ou a un lien
Les tableaux
balise description
table permet de crée un tableau
caption permet de mettre une legend a un tableau
thead defini l'en-tete d'un tableau
tfoot defini la fin d'un tableau
tbody defini le corps d'un tableau
th permet de rajouter un titre a une image ou a un lien
tr defini une ligne de cellule
td defini une cellule du tableau
div permet de grouper des éléments pour leur appliquer un style
Formulaire
balise description
form permet de creer un formulaire
fieldset utilisé afin de regrouper plusieurs contrôles interactifs et ainsi que des étiquettes (label) dans un formulaire
label représente une légende pour un objet d'une interface
input est utilisé pour créer un contrôle interactif dans un formulaire web qui permet à l'utilisateur de saisir des données
legend présente une légende pour le contenu de son élément parent
